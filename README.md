# New Year Greeting Card

A single-page, shareable New Year digital greeting card with:

- Letter-style reveal animation
- Fireworks and confetti effects
- Personalized names via URL parameters
- Music playback control
- WhatsApp share button

## Personalization

Use query parameters in the URL:

- `?to=Alice`
- `?to=Alice&from=Bob`

Example:

`https://your-username.github.io/NewYear2026/?to=Alice&from=Bob`

## Publish on GitHub Pages

1. Push this project to a GitHub repository.
2. Open repository **Settings** -> **Pages**.
3. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/** (root)
4. Save and wait for deployment.
5. Open the generated Pages URL and share it on WhatsApp.

## WhatsApp Message Template

Use this when sharing:

"🎉 I made a special New Year surprise for you! Open this link: <https://your-username.github.io/NewYear2026/>"
